# Informacje o źródłach EPG dla Polski

## Darmowe źródła EPG

### 1. EPG.OVH (aktualnie używane)
- **URL:** https://epg.ovh/plar.xml (10 dni z archiwum)
- **Status:** ✅ DZIAŁA
- **Aktualizacja:** Codziennie
- **Format:** XML
- **Rozmiar:** ~130MB (plar.xml)
- **Uwagi:** Najlepsze darmowe źródło dla Polski

### 2. GitHub iptv-org
- **URL:** https://raw.githubusercontent.com/iptv-org/epg/master/guides/pl/pl.xml
- **Status:** ⚠️ Może być nieaktualne
- **Aktualizacja:** Rzadko
- **Format:** XML

- **Strona:** https://www.globalepgs.info/pl/

### 2. Global Listings
- **Strona:** https://www.globallistings.info/pl/
- **Usługi:** Aktualizacje EPG 24/7
- **Okres próbny:** ❓ NIE POTWIERDZONY - wymaga kontaktu
- **Kontakt:** Skontaktuj się bezpośrednio w sprawie trial

## Rekomendacja

**Aktualnie najlepsze rozwiązanie:**
- Używać **epg.ovh/plar.xml** jako główne źródło
- System automatycznie próbuje różnych źródeł jeśli główne nie działa
- Jeśli potrzebujesz bardziej niezawodnego źródła, skontaktuj się z Global EPGs lub Global Listings w sprawie:
  - Okresu próbnego
  - Ceny dla małych projektów
  - API endpoint dla integracji
